This Java-based console application allows users to browse a list of movies, select showtimes, and reserve seats in a movie theater. The application includes error handling for user input to ensure a smooth booking experience.

## Usage

1. Compile and run the `AppData.java` file.
2. Follow the on-screen prompts to select a movie, showtime, and book seats.
3. Confirm your reservation and enjoy the movie!

## How it Works

- The application utilizes Java's `Scanner` class for user input.
- It maintains a database of movies and their showtimes.
- Users can choose from available showtimes and select the number of seats they wish to book.
- The system calculates the total cost and allows users to confirm their reservation.

## Dependencies

- Java 8 or higher

## Getting Started

1. Clone the repository.
2. Compile and run the `AppData.java` file using your preferred Java IDE or command line.
3. Follow the on-screen instructions to book your seats.
